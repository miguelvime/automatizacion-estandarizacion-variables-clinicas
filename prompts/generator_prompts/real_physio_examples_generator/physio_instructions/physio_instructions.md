# Generación de historias clínicas de texto libre a partir de códigos CIF.

## Objetivo
Generar historias clínicas realistas a partir de códigos CIF usando la experiencia clínica de fisioterapeutas. Estos ejemplos servirán para crear un prompt para un LLM que realiza esta misma función.

## Documentos de referencia
1. **icf_brief_spanish.pdf:** Contiene el significado de cada código. También hay información que no necesitarás, pero te puede ayudar a hacer realista el texto por ejemplo información para cuantificar la severidad de cada código, fuentes de información de las cuales se puede derivar el código (entrevista, examen clínico,etc.)
2. **cif_basics.pdf**: manual de instrucciones de uso de la cif.
3. **complete_icf.pdf**: es el manual completo de la CIF, no es necesaria su consulta pero podéis ver más detalles de los códigos en él.

## Instrucciones
1. Lee el documento cif_basics.pdf.
2. Usa los códigos proporcionados junto con el documento icf_brief_spanish.pdf para inventar 3 historias clínicas diferentes para los mismos códigos. Imagina que son pacientes que ves habitualmente en la clínica, escribe la historia como lo harías al tratar a un paciente, tanto en forma de expresarte como en longitud.
3. Revisa que no haya más códigos de los que aparezcan en los proporcionados. Revisa que el código pueda ser asignado al paciente con la historia generada.

Si tienes dudas o quieres ampliar información puedes acudir al manual complete_cif.pdf.
